Smart Sri Lanka Test

introduction

Smart Sri Lanka is ……. First Initial Stage was Register the Members to Smart Sri Lanka Programme ,But the problem was with all of the members they need to filter out the leadership personality to lead the smart sri lanka hub in every district. Therefore after registration newly generated link was sent to the user to fill out the psychometric test.

the challenge

When we going to cover the whole sri lanka , main problems that we can encounter is internet access and language problem. Therefore in order to make sure users understand the questionnaires better system needs to be support variety of three main languages.

Web Design

Initially we supported two Main languages.When we focus on web design , the board of the members suggested installing KIOSK machine to rural area where the user don’t have access to the internet. With that user requirement in mind we started with wireframes honed in on visual design base elements within ui kit.

Visual Design

Website needs to represent the related questionnaires with more interactivity feedback.The final site did just that. Using more interactive colors to the questionnaires answer point, braking questionnaires into parts,using the icons to represent the visual meaning of the rules all while creating desired hierarchy for the content

Implementation

This project main core element is the Psychometric-Test algorithm . Board kindly sent me the hard copy of the algorithm. With that information I Used Jquery, Css, HTML as frontend technologies to give more interactive feedback for the users.In order to process and save the data and give the user output about their personality i used , Laravel(PHP) Framework to the backend.
